WEB이란 무엇인가

하이퍼텍스트 마크업 랭귀지 (HTML)는 표준 마크업 언어이며 웹 페이지와 웹 애플리케이션을 만들기 위한 것이다. 웹 브라우저는 반응한다 HTML도큐멘트 웹 서버에서 또는 로컬 스토리지 그리고 렌더 그것들은 멀티미디어 웹 페이지 안에 있다

HTML 디스크라입스 구조물 웹 페이지 세먼티셜리 그리고 오리지널리 포함되다 도큐멘트가 나타나는 것에. HTML 오브제들은 HTML 페이지의 블럭을 구축한다
그리고 HTML 컨스트럭쳐, 이미지 또는 오브젝트들 인터랙티브 폼, 아마 다가오다 렌더페이지 안에 있는 것. 그것은 프로비즈 의미한다 구조물을 만드는 도큐멘트 디노팅 스트럭츄럴 세맨틱스 폴
텍스트 서치 애즈 헤딩 파라그래프,리스트,링크,쿼츠 앤 다른 아이템들. HTML 물질들은 델리니티드 바이 태그. 태그를 쓰는 브라켓들
